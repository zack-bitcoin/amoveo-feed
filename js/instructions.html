Amoveo Exchange Instructions

  1. Go to home
  2. Deposit required VEO to exchange account
  3. Wait for 4 confirmations.
  4. Fill all required field:
    • message you want to publish
  5. Click on generate unsigned post request
  6. Copy unsigned post request (whole string) and sign it with the light wallet
  7. Copy signed request from wallet and paste in into field next to publish signed post request and click on this button.
